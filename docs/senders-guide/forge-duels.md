## What is The Forge?

### Competition Mechanics

**The Forge Process:**

1. **Entry Requirements**
   - Own the Spark NFT
   - Stake 1,000+ $SIGNAL
   - Complete founder profile
   - Pass basic verification

2. **Competition Period**
   - 7-14 days typical
   - Public presentations
   - Community Q&A
   - Live debates

3. **Victory Conditions**
   - Highest belief signals
   - Community vote
   - Anchor endorsement
   - Technical merit

<ul>

<p>2. **Invited Duel** (Curated)</p>

</ul>
